Html

História

A criação do HTML se confunde, em partes, com a própria história da internet como conhecemos hoje. Isso porque foi graças ao HTML que a internet se popularizou, permitindo que pessoas de diferentes lugares e níveis de conhecimento pudessem utilizá-la. image resource

Fundador

Em 1990, o cientista Tim Berners-Lee criou o HTML para possibilitar o acesso e a troca de documentos de pesquisa entre cientistas de diversas universidades. É ele o criador do World Wide Web Consortium (W3C) e ganhou vários prêmios devido a sua contribuição para a tecnologia, inclusive o prêmio Turing em 2016, considerado o Nobel da computação. Desde a sua criação, o HTML já passou por algumas modificações e melhorias que aumentaram o seu poder de utilização para os mais diversos tipos de aplicações, desde programação web como a construção de páginas e telas para sistemas, desenvolvimento mobile, e atualmente é inclusive utilizado no desenvolvimento de jogos.

Tags

A estruturação e marcação de uma página HTML é feita por meio de elementos chamados de tags. São elas que permitem a utilização correta e montagem de uma página Web. Elas possuem uma estrutura própria para definição: O sinal de “menor que” marca o início de uma tag, seguida pelo nome de um elemento e o sinal de “maior que” para fechar essa tag. Algumas precisam de uma tag de fechamento, indicada pelo sinal de “barra” (/), que delimita um bloco utilizado. Outras tags possuem auto fechamento.

Estrutura

O trecho de código acima contém a estrutura básica de uma página HTML. Algumas tags existentes nela são:

  • html: indica o início e que o arquivo é do formato HTML.
  • head: tag com informações para o cabeçalho da página, muito usado nos motores de busca para indexação de conteúdo.
  • meta: permite o uso de atributos para configuração da página como formato do texto, compatibilidade entre navegadores e resoluções de tela.
  • title: tag que exibe o título de uma página. body: indica o início do corpo do documento. Tudo o que está depois dessa tag fica visível no navegador.
  • div: mostra uma divisão ou um bloco específico dentro da página. h1: exibe um título em tamanho de destaque.
  • p: tag para a escrita de parágrafos.
